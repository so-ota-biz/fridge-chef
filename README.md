## 💡 コンセプト

冷蔵庫にある食材を有効活用し、食品ロスを削減しながら、毎日の献立を考える手間を省くことを目指したアプリケーションです。AI が食材の組み合わせから、ユーザーの好みや条件に合わせた最適なレシピを提案します。

## 📋 要件定義

### プロジェクトの目的

- **日々の献立決めのストレス解消**：献立決定時間を 30 分から 1 分に短縮
- **食材の有効活用**：冷蔵庫の食材を無駄なく使い切り、食材ロスを削減
- **料理のマンネリ化解決**：AI による多彩なレシピ提案で料理のバリエーションを広げる
- **料理への自信向上**：完成写真の記録を通じて料理の上達を実感し、SNS 投稿も促進

### 対象ユーザー

#### プライマリーターゲット

料理に悩む主婦・主夫層

#### セカンダリーターゲット

一人暮らしの社会人、料理初心者

#### 具体的ペルソナ

- 「今日の夕飯何にしよう…」と毎日悩む人
- レシピサイトを見ても「この食材ない」となる人
- 買い物に行く前に、家にあるもので済ませたい人
- 作った料理を SNS に投稿したい人

### 解決したい課題

1. **献立決めの時間とストレス**：毎日の献立を考える負担を軽減
2. **食材の無駄**：冷蔵庫の食材を使い切れず捨ててしまう問題
3. **料理のマンネリ化**：いつも同じような料理になってしまう
4. **料理への自信不足**：料理のスキルアップを実感できない
5. **SNS 映えする写真**：魅力的な料理写真が撮れない

### システムの特徴・差別化ポイント

- **冷蔵庫ベースの料理提案**：保有食材から最適なレシピを AI が自動生成
- **カテゴリ別アイコン選択**：直感的な操作で簡単に食材を選択可能
- **3 パターンのレシピ提案**：条件に合わせた複数の選択肢を提供
- **調理記録の保存**：作った料理を写真付きで記録し、振り返りが可能
- **AI 生成の完成イメージ**：レシピごとに魅力的な完成写真を自動生成（将来実装：実際の写真との比較機能）

## ✨ 主な機能

### 1. 認証機能

- ユーザー登録・ログイン
- JWT 認証によるセキュアなセッション管理

### 2. 食材選択

- カテゴリ別の食材マスタから選択
- カスタム食材の手動入力

### 3. レシピ生成条件の設定

- ジャンル（和食、洋食、中華など）
- 難易度（超簡単、簡単、普通、難しい）
- 調理時間（15 分以内、30 分以内など）
- 人数（1〜10 人分）

### 4. AI レシピ提案

- 選択した食材と条件に基づいて、AI が 3 つのレシピを生成
- レシピ詳細（材料、手順、調理時間など）の表示
- レシピ画像の自動生成

### 5. 調理記録管理

- 作ったレシピの記録作成
- 評価（星 5 段階）とメモの保存
- 調理記録の一覧表示・詳細確認
- 調理記録の編集・削除

### 6. レスポンシブデザイン

- スマートフォン、タブレット、デスクトップに対応
- Mantine UI によるシンプルな UI/UX
